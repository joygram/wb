숫자로 표현한 심플한 카운터입니다.

카운터를 붙이는 요령입니다.
1. 카운터붙이는 페이지는 확장자를 php로 변경해주셔야 합니다.
2. 아래 코드에 값을 변경하신 다음 페이지의 제일 상단에 붙여주시면 됩니다.
<?
$C_base_dir   = "/home/there/whitebbs/wb"; //관리자도구에 표시된[기본 디렉토리]
$C_auth_perm  = "7555" ;  //현재 페이지의 접근권한
$counter_data = "test1" ; //사용할 카운터 이름
$counter_view_only = "" ; //카운트는 하지않고 디스플레이만 할경우 yes
include("$C_base_dir/lib/whitebbs.inc.php") ;
?>
	$C_base_dir 변수는 관리자도구 처음 화면에 들어가면 [기본 디렉토리]라는 정보가 있는데 그 것을 적어주시면 됩니다. ""안에 잘 넣어주세요.

	$counter_data 는 설치하는 카운터의 이름입니다.

	$counter_view_only를 yes로 넣을경우 카운트는 갱신하지 않고 내용만 보여줍니다. 이경우 카운트는 다른 곳에 붙이고 보기도 다른곳에 붙이고 싶은 경우 사용하시면 됩니다.
3. 카운터모양을 넣고 싶은 곳에 <?=$Row_c['counter']?> 변수를 넣어주시면 됩니다.
