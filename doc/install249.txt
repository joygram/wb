*) 처음 설치
    ///////////////////////////////////
    설치과정: 
    ///////////////////////////////////
    *. 설치하고자 하는 디렉토리를 만든다.
        ex) white
        
    *. 설치파일을 새로만든 디렉토리에 ftp를 이용해 올린다.
        setup.php ( pear, tar, wb_untar 포함)
        setup.pkz
    
    *. 웹브라우져 상에서 setup.php파일을 호출한다.
        자신의 홈페이지 주소/새로만든 디렉토리/setup.php
        
        ex) whitebbs.net/white/setup.php
        
        *NOTE: 이때 화이트보드의 패키치 파일의 묶음이 자동으로 해제가 된다.( unpkg249.done 파일 생성 )
        
    *. 설치/업그레이드 선택 
        새로 설치하는 경우이므로 [설치]를 선택하도록 한다. 
        
        
    *. 사용 데이터베이스 설정(setup_db.php)
        admin/system.ini.php 파일에 사용할 데이터베이스를 저장한다.
        
        파일시스템
        PostgreSQL
        MySQL
        기타등등
            :
            :
        *NOTE: 현재는 파일시스템만 사용할 수 있다. 
        *NOTE: 어떤 DB를 선택하느냐에 따라서 관리자의 설정이 저장되는 곳이 결정된다.
        *NOTE: 다른 DB의 자료가 있는 경우 단순히 사용하는 데이터베이스의 설정을 바꾼다고 해서 자동으로 데이터의 이전 되지 않는다.
            ex) 파일 시스템 -> MySQL 선택시
            
        
    *. 관리자 설정(setup_admin.php)
        아이디
        비밀번호
        별명
        이메일
        홈페이지
        
        *NOTE: 파일시스템을 사용하는 경우라면 
        *NOTE: 회원관리가 제공 된다면 관리자의 기본 설정은 사용여부와 관계없이 회원관리 프로그램에 저장이 된다.
        
    *. 사용 프로그램 설정(setup_program.php)
        화이트보드 패키지 중에서 사용하고 싶은 프로그램을 선택하여 관리자 설정부분등에서 불필요한 정보들을 제공하지 않도록 한다. 
        
        보드   [x]
        카운터 [ ]
        회원   [ ]
        메일러 [ ]
            :
            :
        *NOTE: 기본적으로 하나라도 선택하지 않으면 설치과정을 계속 수행할 수 없다. 
        *NOTE: 이과정이 끝나면 기본 설치과정을 무사히 마친것이다. 그 확인으로 admin/setup250.done 라는 파일이 자동 생성된다.
        

    * 기본 설치과정을 무사히 마치게 된다면 설치프로그램은 화이트보드의 각디렉토리를 생성/권한설정을 자동으로 해준다음 아이디와 비밀번호를 묻는다. 관리자 설정에서 지정한 아이디와 비밀번호를 입력하면 관리자도구로 들어가게 된다.
        
        *NOTE: 만일 앞서 지정한 관리자의 아이디와 비밀번호가 잘못기입되어 확인이 안된다면 비상 조취를 취해야 하는데 그방법은 다음과 같다.
            - admin/setup250.done 파일을 ftp를 이용해 제거한다.
            - 웹브라우져 상에서 다시 setup.php파일을 호출한다. ( setup250.done라는 파일이 없다면 설치 업그레이드 부터 다시 진행이 된다)
